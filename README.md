# ⚠️ [WIP] splitser-to-csv
Simple python script to generate CSVs based on splitser settlement PDFs.
## Quick Set Up
```bash
git clone https://github.com/bram-bolt/splitser-to-pdf.git
cd app
```

### 🇬🇧 English (splitser)
```bash
python3 main.py [file]
```

### 🇳🇱Dutch (WieBetaaltWat)
```bash
python3 main.py [file] -l NL
```
