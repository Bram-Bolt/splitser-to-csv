# ⚠️ [WIP] splitser-to-csv
Simple python script to generate CSVs based on splitser settlement PDFs.
## Quick Set Up
```bash
git clone https://github.com/bram-bolt/splitser-to-csv.git
cd splitser-to-csv
```

> TEMPORARY!
### 🇬🇧 English (splitser)
```bash
python -m app.main [file]
```

### 🇳🇱Dutch (WieBetaaltWat)
```bash
python -m app.main [file] -l NL
```
